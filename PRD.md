## 1. نظرة عامة على المنتج

### 1.1 ملخص المشروع

منصة رانية براهمي هي تطبيق ويب متكامل يُمكِّن المدربة الحياتية **رانية براهمي** من تقديم خدماتها في التحويل الشخصي والوعي الذاتي للنساء الناطقات بالعربية. تجمع المنصة بين موقع تسويقي احترافي، ومحرك حجز ذكي للجلسات، ونظام إدارة تعليمي (LMS)، ودفتر ملاحظات مشفر من طرف إلى طرف.

### 1.2 المشكلة التي تحلها

- **المدربون المستقلون** يفتقرون إلى منصة موحدة تجمع التسويق + الحجز + التعليم في مكان واحد.
- **العميلات** (النساء الناطقات بالعربية) يحتجن إلى تجربة آمنة وراقية لطلب الدعم في رحلة التحول الشخصي.
- غياب الخصوصية التامة في أدوات تدوين الملاحظات التقليدية يُعيق الانفتاح العاطفي.

### 1.3 الأهداف الرئيسية

| الهدف | مؤشر النجاح |
|-------|------------|
| زيادة معدلات الحجز | +30% في أول 3 أشهر بعد الإطلاق |
| تقديم تجربة مستخدم راقية | معدل ارتداد < 40% |
| ضمان أمن البيانات | تشفير AES-GCM لجميع الملاحظات الحساسة |
| دعم الطلب في أي مكان | التوافق مع المناطق الزمنية المختلفة |

- **RTL Support:** دعم الاتجاه من اليمين لليسار (العربية).
- **Error Boundaries:** حماية كل قسم بـ `ErrorBoundary` للعزل في حالة الأخطاء.

### 3.3 محرك الحجز (Booking Engine)

**الأولوية:** عالية جداً ✅

#### أنواع الجلسات والأسعار:

| نوع الجلسة | السعر (TND) | الوصف |
|-----------|------------|-------|
| 📹 **فيديو (VIDEO)** | 150 د.ت | جلسة مرئية خاصة 60 دقيقة |
| 🎙️ **صوتي (AUDIO)** | 100 د.ت | جلسة صوتية خاصة 60 دقيقة |
| 💬 **واتساب (WHATSAPP)** | 60 د.ت | جلسة نصية عبر واتساب |

#### مميزات كل جلسة:
- 60 دقيقة جلسة خاصة ومكثفة
- تسجيل ملاحظات مشفر وآمن
- خطة عمل مخصصة بعد الجلسة
- مواعيد مرنة حسب المنطقة الزمنية للعميلة

#### تدفق الحجز:
1. اختيار نوع الجلسة (VIDEO / AUDIO / WHATSAPP)
2. عرض تفاصيل الجلسة والسعر
3. تأكيد الحجز → إنشاء `Booking` بحالة `PENDING`
4. إعادة توجيه لصفحة الدفع
5. تأكيد الدفع → تحديث حالة الحجز إلى `CONFIRMED`

#### حالات الحجز:
```
PENDING → CONFIRMED → COMPLETED
              ↓
           CANCELLED
```

### 3.5 نظام إدارة التعلم (LMS)

**الأولوية:** متوسطة 🔄

#### نموذج الدورة:

| الحقل | الوصف |
|------|-------|
| `title` | عنوان الدورة |
| `description` | وصف الدورة |
| `basePrice` | السعر الأساسي (TND) |
| `durationMonths` | مدة الدورة (افتراضياً 9 أشهر) |
| `muxAssetId` | معرف المحتوى على Mux (حماية المحتوى) |
| `isPublished` | حالة النشر (true/false) |

#### الوحدات (Modules):

| الحقل | الوصف |
|------|-------|
| `muxPlaybackId` | معرف التشغيل المُوقَّع من Mux |
| `duration` | مدة الفيديو (بالثواني) |
| `orderIdx` | ترتيب العرض |
| `isPreview` | هل متاح للمعاينة المجانية؟ |

#### التسجيل والتقدم:
- `progress`: نسبة إتمام الدورة (0.0 → 100.0%)
- `completedAt`: تاريخ الإتمام
- قيد فريد: (userId, courseId) لمنع التسجيل المكرر

### 3.6 دفتر الملاحظات المشفر (Shadow Work Notes)

**الأولوية:** متوسطة 🔄

> ميزة حساسة تتيح للعميلة تدوين ملاحظاتها الشخصية بشكل آمن تماماً.

#### متطلبات الأمان:

| المعيار | التفاصيل |
|--------|---------|
| **خوارزمية التشفير** | AES-GCM (تشفير من طرف إلى طرف) |
| **التخزين** | `contentData` (نص مشفر) + `iv` (متجه التهيئة) |
| **الوصول** | لا يستطيع أحد قراءة المحتوى — حتى المدير |
| **الاستردادية** | لا يمكن استعادة المحتوى عند فقدان المفتاح |

## 4. المتطلبات غير الوظيفية

### 4.1 الأداء

| المعيار | الهدف |
|--------|-------|
| **First Contentful Paint (FCP)** | < 1.5 ثانية |
| **Time to Interactive (TTI)** | < 3 ثوانٍ |
| **Core Web Vitals** | LCP، FID، CLS ضمن النطاق الأخضر |
| **الصورة الرئيسية** | تحميل أولوي (`priority`) + حجم نسبي |

### 4.2 التصميم وتجربة المستخدم

| المعيار | التفاصيل |
|--------|---------|
| **الهوية البصرية** | فاخرة وراقية (ألوان ذهبية دافئة، خلفية داكنة) |
| **الخط** | Cairo (عربي مدور وعريض) |
| **الاتجاه** | RTL بالكامل (عربي) |
| **الحركة** | Framer Motion + Lenis للتمرير السلس |
| **التجاوب** | Mobile-first — يعمل على جميع الأجهزة |
| **الإمكانية** | معايير WCAG 2.1 AA للتباين والوصول |

### 4.3 الأمان

| المعيار | التفاصيل |
|--------|---------|
| **كلمات المرور** | تشفير `passwordHash` (Bcrypt أو مشابه) |
| **الملاحظات** | AES-GCM (E2E encryption) |
| **CSRF** | الحماية عبر Next.js Server Actions |
| **المسارات** | حماية صفحات الداشبورد بـ `middleware.ts` |

### 4.4 الموثوقية والاستمرارية

| المعيار | التفاصيل |
|--------|---------|
| **قاعدة البيانات** | PostgreSQL (مُستضاف) |
| **فيديو الدورات** | Mux (CDN عالمي + HLS) |
| **Error Boundaries** | عزل الأخطاء لكل قسم في الواجهة |
| **ISR** | إعادة توليد الصفحات كل ساعة تلقائياً |

  - Next.js 15 (App Router + Turbopack)
